A. CSS is an abbreviation for Cascading Style Sheets. It is a stylesheet language used to change the desing of a HTML page.

A. There are 3 methods you can use to use colors in CSS.
  1. Names of simple colors (e.g. green, black, aqua, ...)
    • Simple to use
    • Limited number of colors
  2. Hex color codes (e.g. #FFFFFF(white), #000000(black), ...)
    • The first two digits are for red, the next two are for green and the last two are for blue.
  3. RGB (e.g. rgb(127, 127, 127)(gray))
    • usage: rgb(red brightness value, green brightness value, blue brightness value)
  4. RGBA (e.g. rgb(0, 255, 0, 0.5)(transparent green))
    • RGB+alpha channel

  1. Using the style attribute inside a tag
    • e.g. <p style="border:1px solid black;">
    • Not recommended
  2. Using the style tag inside <head>
    • example:
      <style>
       p{
        border:1px solid black;
       }
      </style>
  3. Making a separate CSS file and linking it to the HTML file
    • example:
      <link rel="stylesheet" type="text/css" href="mycss.css">
